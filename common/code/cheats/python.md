# hex

```bash
addr=200.0.10.1; python -c "
import socket; 
print(socket.inet_aton('$addr'))"
# b'\xc8\x00\n\x01'

addr=200.0.10.1; python -c "
import socket, struct;
print(struct.unpack('<L', socket.inet_aton('$addr'))[0])"
# 17432776

addr=200.0.10.1; python -c "
import socket, struct; 
print(hex(struct.unpack('<L', socket.inet_aton('$addr'))[0])[2:10].upper().zfill(8))"
# 010A00C8
```
