# debugging

reading direction: on arrow end, goto next step; each pipe character (`|`) delimits byte

```gdb
pwndbg> x/32x 0x100000000
                <-|-|-|-|1      <-|-|-|-|2      <-|-|-|-|3      <-|-|-|-|4
0x100000000:    0x55554000      0x00005555      0x557a2020      0x00005555
```
